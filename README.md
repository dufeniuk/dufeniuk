# Hi there, I'm Bohdan 👋

**Software Engineer | Python & Django Developer | CAD-CAM Automation Specialist**

I bridge the gap between heavy industrial manufacturing and modern software engineering. With a solid background as a Production Manager and CNC Specialist, I build custom web applications, advanced database solutions, and business automation tools that streamline operations.

### 🚀 What I do
- 💻 **Backend Development:** Building robust custom business systems (ERP/MRP) and APIs using Python and Django.
- ⚙️ **Industrial Automation:** Translating real-world machining logic into scalable software logic.
- 🏭 **CAD/CAM Engineering:** Creating custom scripts, multi-axis CNC programming, and technical workflow optimization.
- 🔧 **Currently working on:** A comprehensive, Django-based CNC office management system integrating technical calculations with e-commerce features.
